# Improved Neural Artistic Style

Improved neural artistic style based on this paper: https://arxiv.org/abs/1605.04603

# Techniques used in this application
- Layer Weight Adjustment
- Activation Shift
- Correlation Chain
- Using L-BFGS for optimization
